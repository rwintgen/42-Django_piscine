Markdown Syntax

Here is a comprehensive guide to Markdown syntax:

  • Headers: Use # for headers, e.g., # Header 1, ## Header 2
  • Emphasis: Use * or _ for italics, e.g., *italic* or _italic_
  • Bold: Use ** or __ for bold, e.g., **bold** or __bold__
  • Lists: Use - or * for unordered lists, e.g., - Item 1
  • Links: Use [text](url) for links, e.g., [Google](https://www.google.com)
  • Images: Use ![alt text](image_url) for images, e.g., ![Alt Text](image_url)
  • Blockquotes: Use > for blockquotes, e.g., > This is a blockquote
  • Inline Code: Use ` for inline code, e.g., `inline code`
  • Code Blocks: Use ``` for code blocks, e.g., ```code block```
  • Horizontal Rules: Use --- for horizontal rules, e.g., ---
